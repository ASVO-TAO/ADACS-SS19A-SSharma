Galaxia is a publicly available code for generating a synthetic survey out of a theoretical model of the Milky Way. The code outputs a catalog of stars according to user specified color magnitude limits. The present version of the code implements the Besancon analytical model of the galaxy and the isochrones used are from the Padova Stellar Evolution database. A web interface is provided here, the full code can be downloaded from www.galaxia.sourceforge.net .
The code is described in Sharma et al. 2019 . One of the main feature of the code is to generate a continuous distribution of stars over a wide angular area. The web interface is currently limited to tasks that require less than X minutes. For more intensive jobs please consider decreasing the sampling parameter fSample or download and run the code locally on your computer. The output data is in a special binary format which we call EBF (Efficient and Easy Binary Format) and these files are denoted with .ebf extension. See http://ebfformat.sourceforge.net/ for further information on reading writing ebf file. The details of the output quantities are given here.
